1. KEKUATAN DAN KELEMAHAN PENDEKATAN EXISTING
================================================================================

A. EFFICIENTNET-BASED APPROACHES
---------------------------------

Kekuatan:
✓ State-of-the-art performance pada berbagai vision tasks
✓ Compound scaling menghasilkan model efisien
✓ Transfer learning efektif dari ImageNet
✓ MBConv blocks cocok untuk fine-grained features

Kelemahan:
✗ Performa pada kelas minoritas masih suboptimal
✗ Belum mengeksplorasi few-shot learning approach
✗ Memerlukan large dataset untuk training optimal
✗ Traditional classifier (softmax) tidak handle imbalance well

Contoh Penelitian:
- Arora et al. [16]: Ensemble EfficientNet → Acc >90%, tapi kompleks
- Ahmed [17]: EfficientNetB0 + Transfer Learning → Avg 85%, class imbalance
- Xu et al. [33]: Hybrid EfficientNet+Swin → Acc 97%, tapi computational heavy


B. PROTOTYPICAL NETWORK APPROACHES
-----------------------------------

Kekuatan:
✓ Excellent few-shot learning capability
✓ Metric-based learning robust terhadap limited data
✓ Prototype formation cocok untuk class imbalance
✓ Tidak memerlukan resampling atau heavy augmentation

Kelemahan:
✗ Belum banyak applied pada medical imaging
✗ Custom architectures kurang transferable
✗ Tidak menggunakan efficient backbone (EfficientNet)
✗ Limited evaluation pada public DR datasets

Contoh Penelitian:
- Murugappan [34]: DRNet (ProtoNet+Attention) → 99.73%, tapi custom CNN
- Ouahab [26]: ProtoMed → Stable features, tapi not DR-specific
- Quan [37]: Dual-Channel ProtoNet → 85.2%, tapi pathology domain

2. RESEARCH GAP (CELAH PENELITIAN)
================================================================================

GAP 1: Ketiadaan Integrasi EfficientNet-B0 + Prototypical Network
-------------------------------------------------------------------

Temuan dari Literature Review:
├─ 12 paper: EfficientNet variants TANPA Prototypical Network
├─ 8 paper: Prototypical Network TANPA EfficientNet
└─ 0 paper: Integration keduanya untuk DR pada APTOS 2019

Mengapa ini GAP?
- Synergy potential: Efficient features + Few-shot learning
- End-to-end optimization belum dieksplorasi
- Perfect match untuk DR challenges (imbalance, subtle features)


GAP 2: Few-Shot Learning untuk Medical Class Imbalance
-------------------------------------------------------

Traditional Methods (yang sudah banyak):
- Resampling (SMOTE, ADASYN)
- Cost-sensitive learning (class weights, focal loss)
- Data augmentation (geometric, color)

Limitasi:
✗ Synthetic data dapat mengurangi medical integrity
✗ Hyperparameter tuning sensitif
✗ Computational overhead

Few-Shot Alternative (yang belum banyak):
✓ Tidak memerlukan synthetic data
✓ Metric learning naturally handle imbalance
✓ Generalization pada minority classes

Why Gap Exists:
- Few-shot learning jarang di-apply untuk medical imbalance
- Community lebih familiar dengan traditional methods
- Perceived complexity dalam episodic training


GAP 3: Evaluasi Komprehensif Focus Minority Classes
----------------------------------------------------

Existing Patterns:
- Overall accuracy (biased toward majority)
- Weighted F1 (still biased)
- Limited per-class analysis

What's Missing:
- Macro-averaged metrics (equal weight per class)
- QWK untuk ordinal classification
- Detailed minority class performance analysis
- Embedding space visualization

Penelitian Ini:
✓ Macro-Precision, Macro-Recall, Macro-F1
✓ Quadratic Weighted Kappa (Kaggle metric)
✓ Per-class confusion analysis
✓ Focus: Class 1 (370 samples) & Class 3 (193 samples)
