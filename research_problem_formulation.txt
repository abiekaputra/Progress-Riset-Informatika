3. SELECT INTERESTED SUB-AREA
================================================================================

Fokus Penelitian:
Integrasi end-to-end antara EfficientNet-B0 (feature extractor) dengan 
Prototypical Network (classifier) untuk menangani klasifikasi multikelas 
Diabetic Retinopathy pada dataset dengan ketidakseimbangan kelas.

Alasan Pemilihan:
- EfficientNet-B0: Efisien (5.3M params), compound scaling optimal
- Prototypical Network: Few-shot learning cocok untuk class imbalance
- End-to-end training: Joint optimization untuk representasi optimal

4. RAISE RESEARCH QUESTIONS
================================================================================

RQ1: Bagaimana penerapan model hibrida EfficientNet-B0 dan Prototypical 
     Network untuk klasifikasi multikelas Diabetic Retinopathy?

RQ2: Bagaimana pengaruh integrasi Prototypical Network terhadap performa 
     klasifikasi dibandingkan model baseline (EfficientNet-B0 + Softmax)?

Sub-Questions:
- Apakah episodic training meningkatkan performa pada kelas minoritas?
- Bagaimana pola confusion antar tingkat keparahan DR?
- Apakah embedding space yang dihasilkan lebih diskriminatif?

5. FORMULATE OBJECTIVES
================================================================================

Tujuan Utama:
1. Menerapkan model hibrida EfficientNet-B0 + Prototypical Network secara
   end-to-end untuk klasifikasi 5 tingkat keparahan DR

2. Menganalisis pengaruh integrasi Prototypical Network terhadap performa
   klasifikasi, khususnya pada kelas minoritas (Mild & Severe)

Tujuan Spesifik:
- Implementasi preprocessing (Circle Crop, Ben Graham)

- Evaluasi menggunakan macro-averaged metrics & QWK
- Analisis embedding space dan prototype positions
- Perbandingan statistik baseline vs hybrid model

7. DOUBLE CHECK

Validasi Pemilihan Metode:

✓ EfficientNet-B0: Terbukti efektif untuk medical imaging [16][17][28]
✓ Prototypical Network: State-of-the-art few-shot learning [23][34][26]
✓ Integration gap: Tidak ada penelitian yang menggabungkan keduanya
✓ Dataset: APTOS 2019 widely used, class imbalance (ratio 9.35:1)
✓ Evaluation: Comprehensive metrics (macro-avg, QWK) untuk imbalanced data

Potensi Risiko & Mitigasi:
- Risiko: Episodic training lebih lambat
  Mitigasi: GPU acceleration, batch optimization
  
- Risiko: Performa mungkin tidak selalu lebih baik dari baseline
  Mitigasi: Multiple random seeds, statistical testing
  
- Risiko: Overfitting pada kelas minoritas
  Mitigasi: Early stopping, validation monitoring
